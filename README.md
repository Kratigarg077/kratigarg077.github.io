## Features

- **Sticky frosted-glass navbar** — transparent on load, blurs on scroll
- **Animated hero section** — gradient name, pulsing availability badge, floating skill chips
- **About section** — expertise tags, work timeline, education
- **Portfolio section** — Professional work projects, engineering achievements, and open-source GitHub projects
- **Skills section** — categorised skill tags + animated proficiency bars
- **Contact section** — all contact details + mailto contact form
- **SEO optimised** — meta tags, Open Graph, Twitter Card, Schema.org JSON-LD, canonical URL
- **Fully responsive** — mobile-first, tested across all screen sizes
- **Scroll animations** — fade-up on intersection, skill bars animate on scroll into view

## Local Development

No build step needed. Just open the file:

```bash
# Clone the repo
git clone https://github.com/kratigarg077/kratigarg077.github.io.git

# Open in browser
open index.html
# or just double-click index.html
```

---

## Deployment (GitHub Pages)

1. Push `index.html` and `README.md` to the `main` branch of `kratigarg077.github.io`
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Site will be live at `https://kratigarg077.github.io`
